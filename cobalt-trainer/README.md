# Cobalt Trainer

An experimental trainer mod for Enshrouded (v0.7+).  
**Use at your own risk.** Single-player only.

## Features
- Infinite Health / Stamina / Mana
- No fall damage
- Unlimited item durability
- Speed multiplier

## Installation
1. Copy `cobalt-trainer` folder into `<Enshrouded>/Mods/`
2. Requires BepInEx 5.4.22+

## Usage
- Press `F1` to toggle menu
- `F5` – God mode
- `F6` – No clip (experimental)
- `F7` – Speed boost
